# Discord Role Application Bot

Бот для Discord сервера "Остров Orlando bot" для автоматизации системы подачи заявок на получение роли с временным доступом на 7 дней.

## Функционал

- **Кнопка "Подать заявку"** в канале "подать-заявку"
- **Модальное окно** для ввода скриншота/описания нахождения в криминальной фракции
- **Создание закрытой ветки** при отправке заявки (доступ только для заявителя и людей с ролью одобрения)
- **Кнопки "Одобрить" и "Отклонить"** в ветке заявки
- **Автоматическая выдача роли** при одобрении
- **Автоматическое снятие роли** через 7 дней
- **Уведомления** в личные сообщения пользователям

   - Разместит кнопку для подачи заявок

## Как работает

### Для пользователей:

1. Заходите в канал "подать-заявку"
2. Нажимаете кнопку "Подать заявку"
3. В модальном окне вставляете ссылку на скриншот или описание
4. Нажимаете "Отправить"
5. Создается закрытая ветка с вашей заявкой
6. Ждете решения от модераторов

### Для модераторов (роль 1507496836724690944):

1. Получаете доступ к закрытым веткам заявок
2. Просматриваете заявку
3. Нажимаете "Одобрить" или "Отклонить"
4. При одобрении:
   - Пользователю выдается роль 1507496778327396455
   - Роль автоматически снимется через 7 дней
   - Пользователь получает уведомление
5. При отклонении:
   - Ветка удаляется
   - Пользователь получает уведомление

## Технические детали

- **База данных**: JSON файл `role_expirations.json` для отслеживания сроков окончания ролей
- **Проверка сроков**: Каждые 5 минут бот проверяет истекшие роли и снимает их
- **Безопасность**: Только пользователи с ролью 1507496836724690944 могут одобрять заявки

## Troubleshooting

**Бот не отвечает на команды:**
- Убедитесь, что бот имеет необходимые права
- Используйте `/setup` для повторной настройки канала

**Роль не выдается:**
- Проверьте, что роль 1507496778327396455 существует на сервере
- Убедитесь, что бот имеет права на управление ролями

**Ветка не создается:**
- Проверьте, что бот имеет права на создание веток
- Убедитесь, что канал "подать-заявка" существует
